Irelia - Ruby on Rails starter kit
----------------------------------

Irelia is a Rails starter kit enhanced with a Tailwind theme and pre-installed Devise for authentication. It's built to kickstart your Rails application by offering out-of-the-box features like login, registration, email verification, and session management. Additionally, Irelia includes built-in support for teams and organizations, making it easy to add multi-user collaboration to your app. Leveraging Tailwind CSS, it provides a modern and responsive design, streamlining your development process right from the start.

## Getting started

To get started with Irelia, follow these simple steps:

#### 1. Clone the repository

```console
git clone git@github.com:kiqr/irelia.git
```

#### 2. Run the setup script
```console
bin/setup
```

#### 3. Start the development environment
```console
bin/dev
```
